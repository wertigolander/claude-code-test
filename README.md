# Web Calculator

A simple, colorful web-based calculator built with HTML, CSS, and JavaScript.

## How to Use

### Opening the Calculator

1. Download or save the `calculator.html` file to your computer
2. Double-click the file to open it in your web browser
3. That's it — no installation or setup needed!

### Using the Buttons

| Button | What it does |
|--------|-------------|
| **0-9** | Type a number |
| **.** | Add a decimal point |
| **+ − × ÷** | Choose an operation |
| **%** | Modulo (remainder after division) |
| **=** | Calculate the result |
| **AC** | All Clear — reset the calculator |
| **DEL** | Delete the last digit you typed |

### Example: Adding Two Numbers

1. Click **5**
2. Click **+**
3. Click **3**
4. Click **=**
5. The display shows **8**

### Chaining Operations

You can chain operations together. For example: `2 + 3 + 4` — just keep pressing operator buttons and the calculator will compute intermediate results automatically.

### Keyboard Support

You can also use your keyboard:

- **0-9** — Type numbers
- **+ - \* /** — Choose an operation
- **Enter** or **=** — Calculate the result
- **Backspace** — Delete the last digit
- **Escape** — Clear everything

## Learning Notes

This project demonstrates several beginner-friendly web development concepts:

- **HTML** — Page structure and button layout
- **CSS Grid** — Arranging buttons in rows and columns
- **CSS Flexbox** — Centering the calculator on the page
- **JavaScript functions** — Each calculator action is its own function
- **Event handling** — Responding to button clicks and keyboard presses
